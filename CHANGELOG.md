## [0.1.0] - 2024-12-05

### Added
- Multi-step profile wizard
- Program matching algorithm with 4 scoring factors
- AI-powered program explanations using Gemini
- Task management with auto-generation
- Document checklist generation
- Dashboard with match statistics
- Seed data with 16 universities and 20+ programs

### Technical
- Next.js 14 with App Router
- Prisma ORM with PostgreSQL
- Tailwind CSS v4
- Google Gemini API integration

### Known Issues
- Mock authentication (no real user system yet)
- Email templates are generated but not sent
- Some mobile UI quirks on smaller screens
- GPA conversion between scales could be more accurate

### Notes
This is the initial MVP. Core features are working but lots of room for improvement. See DEVLOG.md for detailed development notes.
